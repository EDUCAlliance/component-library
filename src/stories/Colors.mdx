# Colors

This page documents all the color variables defined in the Tailwind CSS theme. These colors are
available as CSS custom properties (variables) and can be used throughout your project for
consistent theming.

## How to use these colors

Every color is available in classic Tailwind classes. So you can use them like this:

```html
<div class="bg-black-primary">
  <p>Hello</p>
</div>
```

You can also use these variables directly in inline styles or CSS-in-JS solutions.
